Unzip

압축을 풀고 싶은 폴더 (/home/user 에 푸는것을 추천함) 로 가서 아래를 실행

(base) cgb3@cgb3:~$  unzip archive.zip -d covid19-ct-scans

여기에서 archive.zip 은 다운받은 파일 이름, covid19-ct-scans는 압축파일을 담을 폴더

def read_nii(filepath):
    '''
    Reads .nii file and returns pixel array
    '''
    ct_scan = nib.load(filepath)
    array   = ct_scan.get_fdata()
    array   = np.rot90(np.array(array))
    return(array)

sample_ct   = read_nii(modifiedlist.loc[0,'ct_scan'])

sample_infe = read_nii(modifiedlist.loc[0,'infection_mask'])
sample_all  = read_nii(modifiedlist.loc[0,'lung_and_infection_mask'])
