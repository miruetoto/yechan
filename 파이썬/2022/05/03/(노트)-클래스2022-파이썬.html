        display(HTML(htmlstr.format(title=self.title,url=self.url,end=self.end)))

클래스 사용법 (암기)

- 선언하는 방법 (=양식문서를 만드는 방법)

class STOOOP: ## 양식 이름 설정 
    title = '학교폭력' ## 양식에 포함된 변수1 
    url = 'https://github.com/guebin/IP2022/blob/master/_notebooks/2022-05-07-stop1.jpeg?raw=true' ## 양식에 포함된 변수2 
    end = '멈춰~~~~' ## 양식에 포함된 변수3 
    def stop(self): ## 양식에 포함된 함수1 = 메소드1 
        htmlstr = "<p> {title} </p> <img src='{url}'> <p style='color:black;'> {end} </p>" 
        display(HTML(htmlstr.format(title=self.title,url=self.url,end=self.end)))
  • 규칙1: 메소드(=class안에 정의된 함수)의 첫번째 인자는 무조건 self
  • 규칙2: 메소드에서 class안에서 정의된 변수를 사용하려면 self.변수이름과 같은 꼴로 쓴다.
    • 즉 self.start, self.img, self.end 와 같은 방식으로 써야한다.
    • 가끔씩 self 자리에 KBSHWP.start, KBSHWP.img, KBSHWP.end 와 같이 클래스 이름을 쓰기도 한다.

- 성능1 사용법: 클래스에서 인스턴스를 생성하는 방법

stop1 = STOOOP()

  • 클래스 이름을 쓰고 콘텐츠를 구체화시키는 과정에서 필요한 입력1, 입력2를 ()에 넣는다.
  • 현재는 따로 입력이 없으므로, 그냥 KBSHWP 이후에 입력을 비워둔다. 즉 KBSHWP()로 생성

        print(self.a)

  • 기능1: imgurl2에 저장된 그림을 출력
  • 기능2: '당신은 이 그림을 n번 보았습니다' 출력, 여기에서 $n$은 그림을 본 횟수

class Klass4:
    n=1 
    url= url1
    def show(self):
        display(HTML("<img src='{url}'> <p> 당신은 이 그림을 {n}번 보았습니다.".format(url=self.url,n=self.n))) # 반복실행하면 횟수증가
        self.n=self.n+1

a=Klass4()

a.show()

당신은 이 그림을 2번 보았습니다. </div> </div> </div> </div> </div>
