인덱싱고급(스트라이딩)

- 스트라이딩 [start:stop:step]

lst = list('abcdefgh')
lst

['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h']
lst[0:8:2]

['a', 'c', 'e', 'g']

- 생략

lst[0::2]

['a', 'c', 'e', 'g']
lst[:8:2]

['a', 'c', 'e', 'g']
lst[::2]

['a', 'c', 'e', 'g']

- 예제: 짝수/홀수의 원소 추출

lst[::2] # 1,3,5,7,... 

['a', 'c', 'e', 'g']
lst[1::2] # 2,4,6,8... 

['b', 'd', 'f', 'h']

- 기본설정은 step=1 $\to$ step=2를 넘어서 step=-1이라면?

lst[::-1]

['h', 'g', 'f', 'e', 'd', 'c', 'b', 'a']
  • 처음과 끝은 생략되어있고.. step=-1이니까 reverse랑 같은 기능이네

(reverse)와 비교

lst1= lst 
lst2= lst

print(lst2[::-1])

['h', 'g', 'f', 'e', 'd', 'c', 'b', 'a']

- 사실 -step은 조금 까다롭다.

(예제) 처음과 끝을 생략하지 않고 아래의 코드와 동일한 코드를 만들어보자.

lst[::-1]

['h', 'g', 'f', 'e', 'd', 'c', 'b', 'a']

(풀이)

결국 lst[?:?:-1] 의 꼴에서 적당히 ?의 값을 채우면 된다.

lst[-1::-1] # 일단 첫시작은 젤 끝이 원소니까 

['h', 'g', 'f', 'e', 'd', 'c', 'b', 'a']
lst[-1:0:-1] # 앗 마지막 인덱스 포함안되는거!

['h', 'g', 'f', 'e', 'd', 'c', 'b']
lst[-1:-1:-1] # 이게 아니잖아..

컴프리헨션 고급 (if문이 포함된 컴프리헨션)

- 예제: 제곱수중에서 12로 나누어 떨어지는 수만 만들고 싶다.

  • 제곱수: 1,4,9,16,25,...
  • 12로 나누어 떨어지는 수

(풀이1)

lst = [] 
for i in range(1,101): 
    if (i**2 % 12 ==0):
        lst.append(i)    

lst

[6, 12, 18, 24, 30, 36, 42, 48, 54, 60, 66, 72, 78, 84, 90, 96]

(풀이2)

[i for i in range(1,101) if i**2 % 12 ==0] 

[6, 12, 18, 24, 30, 36, 42, 48, 54, 60, 66, 72, 78, 84, 90, 96]

(풀이3)

list(range(101))[6::6] # 위의 풀이에 비해서 가독성도 별로고 딱히 더 짧은것도 아님

[6, 12, 18, 24, 30, 36, 42, 48, 54, 60, 66, 72, 78, 84, 90, 96]

함수고급 (조건부리턴) 

def test(a):
    if a % 2 == 0:
        return 'even'
    else:
        return 'odd'

[test(a) for a in range(1,11)]

['odd', 'even', 'odd', 'even', 'odd', 'even', 'odd', 'even', 'odd', 'even']
def test(a): 
    return 'even' if a%2 ==0 else 'odd'

[test(a) for a in range(1,11)]

['odd', 'even', 'odd', 'even', 'odd', 'even', 'odd', 'even', 'odd', 'even']
