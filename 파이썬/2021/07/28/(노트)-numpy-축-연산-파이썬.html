(노트) numpy 축 연산

Jul 28, 2021 • 신록예찬 • 3 min read

파이썬

  • 2차원
  • 3차원 
import numpy as np

2차원

- 연습용배열

a2=np.array([[1,2,3],[4,5,6]])
a2

array([[1, 2, 3],
       [4, 5, 6]])

a2는 $2 \times 3$ 행렬이다.

a2.shape

(2, 3)

- 예제1

axis=0으로 연산하면 (2,3)중 첫번째 차원이 삭제된다.

  • 따라서 결과는 길이가 $3$ 인 벡터가 나옴
  • 삭제된 차원값 = 2 $\to$ 결과의 각 원소는 2개의 숫자를 더한 것임 
np.sum(a2,axis=0)

array([5, 7, 9])

- 예제2

axis=1 으로 연산하면 (2,3)중 두번째 차원이 삭제된다.

  • 따라서 결과는 길이가 2인 벡터가 나옴
  • 삭제된 차원값 = 3 $\to$ 결과의 각 원소는 3개의 숫자를 더한 것임 
np.sum(a2,axis=1)

array([ 6, 15])

3차원

- 연습용배열

a3=np.array([[[1,2,3],[4,5,6]],[[11,12,13],[14,15,16]],[[21,22,23],[24,25,26]],[[31,32,33],[34,35,36]]])
a3

array([[[ 1,  2,  3],
        [ 4,  5,  6]],

       [[11, 12, 13],
        [14, 15, 16]],

       [[21, 22, 23],
        [24, 25, 26]],

       [[31, 32, 33],
        [34, 35, 36]]])

a3의 차원은 $4\times 2 \times 3$

a3.shape

(4, 2, 3)

- 예제3

axis=0 으로 연산하면 (4,2,3) 중 첫번째 차원이 삭제된다.

  • 따라서 결과는 (2,3) array
  • 삭제된 차원값 = 4 $\to$ 결과의 각 원소는 4개의 숫자를 더한 값임
### 원래 array
array([[[ 1,  2,  3],
        [ 4,  5,  6]],

       [[11, 12, 13],
        [14, 15, 16]],

       [[21, 22, 23],
        [24, 25, 26]],

       [[31, 32, 33],
        [34, 35, 36]]])

### 계산후에는 대충 이런모양
array([[1+?, 2+?, 3+?],
       [4+?, 5+?, 6+?]])    

### 좀 더 구체적으로 쓰면
array([[1+11+21+31, 2+12+22+32, 3+13+23+33],
       [4+14+24+34, 5+15+25+35, 6+16+26+36]])
np.sum(a3,axis=0)

array([[64, 68, 72],
       [76, 80, 84]])

- 예제4

axis=1 으로 연산하면 (4,2,3) 중 두번째 차원이 삭제된다.

  • 따라서 결과는 (4,3) array
  • 삭제된 차원값 = 2 $\to$ 결과의 각 원소는 2개의 숫자를 더한 값임 
### 원래 array
array([[[ 1,  2,  3],
        [ 4,  5,  6]],

       [[11, 12, 13],
        [14, 15, 16]],

       [[21, 22, 23],
        [24, 25, 26]],

       [[31, 32, 33],
        [34, 35, 36]]])

### 계산후에는 대충 이런모양 
array([[ 1+?,  2+?,  3+?],
       [11+?, 12+?, 13+?],
       [21+?, 22+?, 23+?],
       [31+?, 32+?, 33+?]])

### 좀 더 구체적으로 쓰면
array([[ 1+4,  2+5,  3+6],
       [11+14, 12+15, 13+16],
       [21+24, 22+25, 23+26],
       [31+34, 32+35, 33+36]])
np.sum(a3,axis=1)

array([[ 5,  7,  9],
       [25, 27, 29],
       [45, 47, 49],
       [65, 67, 69]])

axis=2로 연산하면 (4,2,3) 중 세번째차원이 삭제된다.
