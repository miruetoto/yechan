lambda

- 예제1: 사용방법

f= lambda x,y,z: x+y+z ## 람다 입력:출력 

f(2,3,4)

9

- 예제2: 디폴트입력가능

x= (lambda a='fee', b='fie', c='foe': a+b+c) 

x('wee')

'weefiefoe'

- 예제3: 람다들의 리스트 가능

   lambda x: x**4]

for f in L: 
    print(f(2))

- 예제4: 람다들의 딕셔너리가능

dct={'func1': (lambda x: x+1),

- 예제5: 조건부출력

- 예제6: lambda expression 자체를 출력가능

def action(x): 
    return (lambda y: x+y)

act = action(99) # act는 99+y 를 수행하는 함수
act2 = action(98) # act2는 98+y 를 수행하는 함수
  • action은 마치 함수를 만드는 함수같다.. 
print(act(2))
print(act2(2))

101
100

- 예제7: 예제6의 발전

action = (lambda x: (lambda y: x+y))

act2 = action(98)

print(act(2))
print(act2(2))

101
100

괄호를 생략하여 더 간단히하면

action = lambda x: lambda y: x+y

act2 = action(98)

map

- 예제1: 사용방법

def inc(x): return x+1 

list(map(inc, [1,2,3,4]))

[2, 3, 4, 5]

- 예제1의 변형(람다사용)

list(map(lambda x: x+1, [1,2,3,4]))

[2, 3, 4, 5]
  • 함수명을 쓰는자리에 lambda로 선언한 오브젝트 자체를 전달 $\to$ 코드가 간단하다.

- 예제2: map과 리스트컴프리헨션 비교

함수선언

f = lambda x: 'X' in x 

f('X1'),f('X2'),f('Y2'), f('Y3')

(True, True, False, False)

map

list(map(f, ['X1','X2','Y3','Y4']))

[True, True, False, False]

리스트컴프리헨션과 비교

[f(x) for x in ['X1','X2','Y3','Y4']]

[True, True, False, False]

- 예제3: 두개의 입력을 받는 함수와 map, 리스트컴프리헨션과 비교

함수소개

pow(2,3)

8

map

list(map(pow, [2,2,2,3,3,3],[0,1,2,0,1,2]))

[1, 2, 4, 1, 3, 9]

리스트컴프리헨션과 비교

[pow(x,y) for x,y in zip([2,2,2,3,3,3],[0,1,2,0,1,2])]

[1, 2, 4, 1, 3, 9]

- 예제4: map은 (동일한 함수,동일하지 않은 입력)에서 가능

L=[lambda x: x+1, lambda x: x+2, lambda x: x+3]

list(map(L,[100,200,300])) ## 에러난다..

---------------------------------------------------------------------------
TypeError                                 Traceback (most recent call last)

----> 1 list(map(L,[100,200,300])) ## 에러난다..

리스트컴프리헨션은 가능함

- 종합: 리스트컴프리헨션과 비교하면, (1) 반복인덱스를 쓰지 않는 장점이 있는 반면 (2) 좀더 제약적으로 사용할수밖에 없다는 단점이 있음

filter

- 예제1

list(filter(lambda x: x<3, [1,2,3,4]))

[1, 2]

리스트컴프리헨션으로 하면

[x for x in [1,2,3,4] if x<3]

[1, 2]

위의 코드를 이해하기 어렵다면 아래를 관찰하라.

for x in [1,2,3,4]:
    if x<3: print(x)

1
2

filter의 이해

filter는 은근히 유용하다. (리스트 컴프리헨션이 너무 좋은것일 뿐..) 예제에서 배운

list(filter(lambda x: x<3, [1,2,3,4]))

는 아래와 같은 코드이다.

f=lambda x: x<3 
x=[1,2,3,4]

fx=list(map(f,x))
fx

[True, True, False, False]
import numpy as np
filtered=np.array(x)[fx]
filtered

array([1, 2])
filtered.tolist()

[1, 2]
