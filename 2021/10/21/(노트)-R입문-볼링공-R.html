(풀이1)

a=c(1,3,2,3,2)

for (i in 1:5){
    for (j in 1:5){
        if (a[i]!=a[j]) count=count+1
    }
}
count
16
  • ? 중복이 계산되어서 이상한 값이 나왔다..

- 아래와 같이 수정하면 된다.

count=0
for (i in 1:5){
    for (j in 1:5){
        if ((a[i]!=a[j])&(i>j)) count=count+1
    }
}
count
8
