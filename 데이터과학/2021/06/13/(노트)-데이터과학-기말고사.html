# 1.

아래와 같은 모형에서 100개의 sample $(x_i,y_i)$를 관찰하였다고 하자.

$$y_i=\beta x_i+\epsilon_i, \quad \epsilon_i \sim N(0,1)$$ 

import numpy as np
import matplotlib.pyplot as plt
np.random.seed(0)
x=np.linspace(0,10,100)
y=2*x+np.random.normal(size=100)
plt.plot(x,y,'.')

(1) 오차제곱합의 최소값을 활용해 $\beta$를 추정하고자 할때 적절한 손실함수를 쓰시오.

(2) 최대우도함수의 최대값을 활용해 $\beta$를 추정하고자 할때 적절한 손실함수를 쓰시오. 

`(3)` (1)과 (2)를 비교하고 

import tensorflow as tf

m1 = tf.keras.models.Sequential()

m1.add(tf.keras.layers.Dense(1,name='outputlayer',activation='linear'))
m1.compile(loss='mse')
