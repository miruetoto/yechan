나노에디터

- 파일열기

sudo nano /etc/apt/sources.list

- 읽기전용 파일만들기: 파일을 만드는것도 파일 여는방법과 동일함. 즉 아래와 같이 하면 exam.txt가 현재 폴더에 있다면 열고아니면 (읽기전용으로) 만든다.

sudo nano /home/cgb/Desktop/exam.txt

- 읽기전용이 아닌 일반파일 만들기

nano /home/cgb/Desktop/exam.txt

- 나노에디터종료: 컨트롤+X

- 파일저장: 컨트롤+O

- 찾기: 컨트롤+W

- 되돌리기: 알트+U

ssh

- 처음에 ssh를 연결하기위해서는 연결당하는 컴퓨터에 가서 아래를 실행해야 한다.

sudo apt install openssh-server

sublime

- 찾아바꾸기: 컨트롤+h

- 화면분할: 옵션+커맨드+2,3,4 ...

파일생성, 파일삭제

- 파일생성

cat > sample.txt 
# Note: 파일생성이후에는 아무것이나 치고 (예를들어 `asdf`) `컨트롤+C`를 눌러서 빠져나온다.

- 폴더삭제

rm -r folderName

- 휴지통삭제

rm -rf .local/share/Trash/files/*
