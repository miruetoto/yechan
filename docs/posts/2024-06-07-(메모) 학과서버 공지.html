  • 개인 원격 서버 구축 및 관리 능력 배양.

3. 강의 범위

A. 기본과정

학과서버를 사용할 수 있는 최소한의 지식 (이건 꼭 강의할겁니다.)

1. 리눅스기초

  • 디렉토리 관리, 리소스 관리, 터미널의 이해, tmux 사용법, vi 사용법, 애플리케이션 설치, 포트 관리, 프로세스 종료.
  • ssh 를 이용한 원격서버 접속, ssh 포트 우회하기

2. 아나콘다를 이용한 가상환경 구축

  • 가상환경의 이해, 가상환경 생성 및 삭제
  • 아나콘다를 이용한 PyTorch / Huggingface 실습 환경 생성.

3. 원격 서버 접속:

  • 주피터랩, RStudio, VSCode 원격 서버 생성 및 접속.

B. 확장과정

이건 학생들이 수강속도가 매우 빠르거나.. 제가 여력이 된다면 추가적으로 진행할 가능성이 있는 주제입니다…

1. 버전 관리 시스템의 기본 개념과 Git의 중요성.

  • 기본 명령어 (clone, commit, push, pull 등) 및 브랜치 관리.
  • 협업에서의 Git 사용과 공통 문제 해결 전략.

2. 클라우드기반 개발환경의 활용

  • 클라우드기반 개발환경의 소개 및 사용: Colab, AWS, Kaggle 노트북, Codespace
  • 데이터 과학 프로젝트에 클라우드 기반 개발환경을 이용하는 방법.

3. 패키지 충돌과 가상 환경

  • 패키지 관리와 패키지 충돌의 원인 및 해결 방법.
  • Anaconda와 Docker를 사용한 가상 환경 구성.
  • 가상 환경에서의 데이터 과학 프로젝트 관리.

4. Quarto+GitHub을 이용한 출판

  • Quarto와 GitHub의 연동을 통한 과학/기술 출판의 기초.
  • 웹사이트, 블로그, 대시보드, 기사, 프레젠테이션 작성 방법.
  • Quarto로 작성된 컨텐츠의 GitHub 배포 및 관리.

5. GitHub을 이용한 R/Python 패키지 배포 및 테스트

  • GitHub을 통한 R/Python 패키지의 개발 및 배포 과정.
  • 타인이 배포한 패키지를 테스트하고 기여하는 방법.
  • 버전 관리 및 커뮤니티와의 협업을 통한 지속적인 개선 전략.

4. 수강 대상

  • 고성능 GPU를 이용하여 쾌적하고 안정적인 학습환경을 구축하고 싶은 학생. (= 코랩보다 더 좋은 환경에서 공부하고 싶은 학생)
