folium.plugins.HeatMap

- 지금까지는 데이터 시각화라고 볼 수 없었음.

- HeatMap은 폴리움에서 데이터 시각화에 필요한 기본적 도구임

import numpy as np
data = (
    np.random.normal(size=(100, 3)) + np.array([[28, 77, 1]])
).tolist()
m = folium.Map([28,77],tiles='OpenStreetMap', zoom_start=6,scrollWheelZoom=False)
folium.plugins.HeatMap(data).add_to(m)
m

folium.plugins.HeatMapWithTime

- 입자의 위치

$$ \begin{cases} x_i = x_{i-1}+\epsilon_{i} \\ y_i = y_{i-1}+\eta_{i} \end{cases} $$

단 $(x_0,y_0)=(35.836603154469884, 127.11406532500682)$

t0 = np.array([[35.836603154469884, 127.11406532500682]]*3000)
walk = np.random.normal(size=(3000, 2, 200)) * 0.0001 

t0[:5] # 시점 0 

(t0+walk[:,:,0])[:5] # 시점1 

walk[:5,:,0]

walk[:5,:,1]

walk[:5,:,0]+walk[:5,:,1]

walk[:5,:,:2].sum(axis=2)

(t0+walk[:,:,:2].sum(axis=2))[:5] # 시점2

data = [(t0+walk[:,:,:i].sum(axis=2)).tolist() for i in range(100)] # 랜덤한 확산.. 

m = folium.Map([35.836603154469884, 127.11406532500682], 
                tiles='OpenStreetMap', 
                zoom_start=17,
                scrollWheelZoom=False)
hmwtPlugin = folium.plugins.HeatMapWithTime(data,radius=7)
hmwtPlugin.add_to(m)
m

import pandas as pd

df

lst=[df.assign(Year=list(map(lambda x: x.split('/')[-1],df.Date))).\
     groupby('Year').\
     pipe(list)[i][1].\
     iloc[:,[1,2,3]].\
     to_numpy().tolist() for i in range(2017-1965)]

Step2: folium 

hm = folium.Map([0,0], tiles='OpenStreetMap', zoom_start=1,scrollWheelZoom=False)
#Add the Heatmap data to HeatMapWithTime plugin        
hmwtPlugin = folium.plugins.HeatMapWithTime(lst,index=list(range(1965,2017)),radius=6)
hmwtPlugin.add_to(hm)
