- 내용요약

  • 여름 $\to$ 수영장 $\to$ 소아마비
  • 여름 $\to$ 아이스크림
  • 아이스크림과 소아마비는 상관관계가 높다: 아이스크림 성분중에 소아마비를 유발하는 유해물질이 있을 것이다(?)

Toy example

- 교재의 예제상황은 예를들면 아래와 같다.

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

- 아이스크림 판매량 = 20 + 온도 $\times$ 2 + $\epsilon$

np.random.seed(1)
temp=np.array([-10.2, -5.2, 0.1, 10.1, 12.2, 14.7,
               25.4, 26.8, 28.9, 35.1, 32.2, 34.6] )
ϵ1=np.random.normal(size=12,scale=5)
icecream = temp * 2 + 20 + ϵ1

plt.plot(temp,icecream,'.')

[<matplotlib.lines.Line2D at 0x7f7bc23213d0>]
  • 온도와 아이스크림판매량의 산점도

- 질병 = 30 + 온도 $\times$ 0.5 + $\epsilon^*$

np.random.seed(2)
ϵ2=np.random.normal(size=12,scale=5)
disease= temp * 0.5 + 30 +ϵ2 

plt.plot(temp,disease,'.')

[<matplotlib.lines.Line2D at 0x7f7bc1ac7310>]
  • 온도와 소아마비의 산점도

- 아이스크림과 질병의 산점도를 그려보자.

plt.plot(icecream,disease,'.')

[<matplotlib.lines.Line2D at 0x7f7bc19c60d0>]
  • 양의 상관관계에 있다.

- 아이스크림 중 어떠한 물질이 소아마비를 일으키는 것이 분명하므로 (인과성이 분명해보이니까) 아래와 같은 모형을 세우자. <-- 여기서부터 틀렸음

$${\tt disease}_i = \beta_0 + \beta_1 {\tt icecream}_i + \epsilon_i, \quad {\bf for} ~ i=1,2,\dots, 12 $$

- 적절한 $\beta_0$와 $\beta_1$을 추정하면 우리는 아이스크림과 소아마비의 관계를 알 수 있다. <-- 틀린주장

  • 틀린모형
  • 도데체 우리가 뭘 잘못했는가?

- 두 변수 사이의 상관관계가 있어도 원인이 다른변수에 숨겨진 경우가 있음.

  • 예제의 논리로 치면 아이스크림과 익사자도 상관계수가 높을것임.
  • 그렇다면 아이스림을 먹으면 물에 빠져죽을것이다? 라고 주장할 수 있나?
  • 사실 기온이 숨겨진 원인이고 기온이 증가하면 아이스크림 판매량도 증가하고 폭염때문에 익사사고율도 높아지는 관계임

- 아래와 같은 예제를 생각하자.

  • 인구수 $\to$ 교회
  • 인구수 $\to$ 범죄건수
  • 지역별 교회와 범죄건수를 살펴보면 상관관계가 높게 나올것임

- 교회를 지으면 범죄건수가 증가한다?

  • 사실 그렇지 않을것임.
  • 인구수가 비슷한 도시끼리 묶어서 비교해보면 교회수와 범죄건수는 상관없을것임!!
