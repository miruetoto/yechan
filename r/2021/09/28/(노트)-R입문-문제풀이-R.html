자료 $x_1,x_2,\dots,x_{100}$에 대하여 다음 과정을 순서대로 시행하였다.

(가) 처음 두 수 $x_1$과 $x_2$의 평균을 구한다.

(나) $x_3$을 추가하여 $x_1,x_2,x_3$의 평균을 구한다.

(다) $x_4$를 추가하여 $x_1,x_2,x_3,x_4$의 평균을 구한다.

...

$x_{100}$을 추가하여 $x_1,x_2,\dots,x_{100}$의 평균을 구한다.

위의 과정을 시행할 결과, $x_1$과 $x_5$의 평균이 5이고, 자료 하나가 추가될때 마다 평균이 1씩 증가하였다. 이때 $x_{100}$의 값은?

(1) 194

(2) 196

(3) 198

(4) 200

(5) 202

(풀이)

x<-c()
x[1]=5
x[2]=5
for(n in 3:100){
    temp_ = mean(x[1:(n-1)])+1 # 1~ (n-1)의 평균 + 1 = temp =1~n 의 평균 
    sum_ = temp_ *n # 1~n의 합 
    x[n] = sum_ -sum(x[1:(n-1)])
}

자연수 $n$에 대하여 점 $P_n$을 다음 규칙에 따라 정한다.

(가) 점 $P_1$의 좌표는 $(1,1)$이다.

(나) 점 $P_n$의 좌표가 $(a,b)$일 때,

  • $b<2^a$이면 점 $P_{n+1}$의 좌표는 $(a,b+1)$이고
  • $b=2^a$이면 점 $P_{n+1}$의 좌표는 $(a+b,1)$이다.

점 $P_n$의 좌표가 $(10,2^{10})$일 때, $n$의 값은?

(1) $2^{10}-2$

(2) $2^{10}+2$

(3) $2^{11}-2$

(4) $2^{11}$

(5) $2^{11}+2$

(풀이) for, if, else, break, ==, &

a=c()
b=c()
a[1]=1
b[1]=1
for(n in 1:2**12){
    if ((a[n]==10)&(b[n]==2^10)){
        print(n)
        break
    }
    if (b[n]<2^(a[n])){
        a[n+1]=a[n]
        b[n+1]=b[n]+1
    }else{
        if(b[n]==2^(a[n])){      
        a[n+1]=a[n]+1
        b[n+1]=1
        }
    }
}
